# bharatnatyam-app
## Database Setup

1. Clone the repo
2. Create a PostgreSQL database named `Bharatnatyam`
3. Restore the schema and data:
   ```bash
   psql -U postgres -d Bharatnatyam -f bharatnatyam.sql

## Running code
1. Create and activate a virtual environment
2. run command:- pip install -r requirements.txt
3. run command:- python app.py
